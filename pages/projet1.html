Projet 1 - Site Vitrine Agence de Voyage

C3 C4 C6

1. Contexte

C'était un projet en solo de quelques semaines où le but était de réaliser un site vitrine pour une agence de voyage fictive présentant rapidement l'agence et les différents voyages proposés par l'agence en le rendant dynamique, visuellement esthétique et simple d'utilisation.

Type de projet

Projet Personnel / PPE

Durée

Quelques semaines

Équipe

Projet solo

Environnement

Développement web

2. Objectifs

Les objectifs principaux de ce projet étaient :

  • Créer un site vitrine attractif et professionnel pour une agence de voyage fictive
  • Présenter clairement l'agence et son offre de services
  • Afficher les différents voyages avec toutes les informations nécessaires (nom, destinations, prix, description)
  • Garantir une interface dynamique et visuellement agréable
  • Assurer une navigation simple et intuitive pour les utilisateurs
  • Rendre le site responsive pour une consultation sur tous les appareils

3. Travail réalisé

Phase de conception

Il a d'abord fallu, comme tout projet de site web, imaginer à quoi allait ressembler le site dans sa structure basique avec notamment l'agencement des pages et comment allaient être réparties les informations importantes sur l'agence en elle-même, mais également les informations indispensables sur les voyages proposés par l'agence comme le nom du voyage, les destinations, le prix et une rapide description.

Développement front-end

Ensuite il a fallut implémenter toutes les pages et les designs, les connecter entre elles tout en le rendant le plus ergonomique possible
